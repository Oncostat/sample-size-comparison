Required Events in Time‑to‑Event Designs

Numerical comparison across East, nQuery, and R

This short note complements the comparison of sample size calculations by focusing specifically on the required number of events for time‑to‑event designs. Using the same scenarios and assumptions, we report the numerical results obtained from East, nQuery, and R implementations, highlighting how closely the tools agree—both for fixed designs and group‑sequential settings. The tables and plots below summarize these event‑based results.
